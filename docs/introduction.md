强大易用的Java 后台开发脚手架。

# 快速开始

```shell
docker run -d --name halo -p 8090:8090 -v ~/.halo2:/root/.halo2 halohub/halo:2.19
```

以上仅作为体验使用，详细部署文档请查阅：https://docs.chenyunlong.cn/getting-started/install/docker-compose
